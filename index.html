What is box sizing?

The box-sizing property allows us to include the padding and border in an element's total width and height.

Different values of box-sizing are as follows:-

content-box:

It is the default value.he width and height properties includes only the content. Border and padding are not included

border-box:

The width and height properties includes content, padding and border

What is animation delay?

The animation-delay property specifies a delay for the start of an animation.The animation-delay value is defined in seconds (s) or milliseconds (ms).The default value is 0s.We can also put the animation delay values to a negative value.This will appear as if it is already being played for some seconds.

What is grid-template-columns used for?

The grid-template-columns property specifies the number (and the widths) of columns in a grid layout.

Which property you will use to merge cells vertically in grid?

The Css grid column property is used to merge the cells vertically in a grid.We specify the start value of the cell we are suppose to span and end value as the number of cells to which it has to be span.

Syntax:

grid-column:grid-column-start/grid-column-end;

Which property you will use to merge cells horizontally in grid?

he Css grid row property is used to merge the cells vertically in a grid.We specify the start value of the cell we are suppose to span and end value as the number of cells to which it has to be span.

Syntax:

grid-row:grid-row-start/grid-row-end;

what is the difference between display none and visibility hidden?

'visibility:hidden' hides the element, but it still takes up space in the layout. Whereas, 'display:none' removes the element from the document. It does not take up any space.
